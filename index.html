Frequently Asked Questions

Is Bous completely free to use?

Yes! Bous is 100% free with no ads, no subscriptions, and no paywalled features.

How does the VLC / external media player export work?

When you select a stream or title, Bous creates a standard XSPF playlist file right in memory and hands it off to your browser's default media player using the downloads.open permission.

Does Bous collect my browsing history?

Absolutely not. Bous operates locally inside your browser (`chrome.storage.local`) and only fetches metadata when you actively interact with the extension sidebar.

How do I install the extension?

Click the "Add to Chrome" button at the top of this page to visit the official Chrome Web Store listing, then click "Add to Chrome" to install it instantly.
