Connecting MDBList to Bous

Bous uses MDBList to fetch dynamic ratings, ranking data, and highly curated lists. Follow this quick guide to generate your API key and sync your account in seconds.

Step-by-Step Guide

Step 1 Sign up at MDBList
Go to the MDBList Website and log into your account, or sign up for a free account if you do not have one yet.
Step 2 Go to Preferences
Once you are logged in, navigate directly to your account Preferences page from the main site menu.
Step 3 Find the API Key Section
Scroll down the preferences page until you find the dedicated API Key section.
Step 4 Generate and Copy
Click the button to generate your unique API key (or reveal it if it is already generated). Copy this key directly to your clipboard.
Step 5 Paste into Bous
Open the Bous extension in your browser, open the settings panel (gear icon), and paste your key into the MDBList API Key field. You're all set!
