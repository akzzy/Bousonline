Integration Guide

Connecting TMDB to Bous

Bous uses The Movie Database (TMDB) to provide rich metadata, cast information, and high-quality images. Follow this simple guide to get your free API key and unlock the full experience in less than two minutes.

Step-by-Step Guide

Step 1 Sign up at TMDB
Go to themoviedb.org and complete the standard signup. Verify your email address. It only takes a couple of minutes!
Step 2 Request an API Key
Once logged in, click your profile icon, go to Settings → API → Request an API Key. Click on the Developer plan. It is completely free forever.
Step 3 Fill the Application Form

This is where some people get stuck. Here is exactly what to put in the tricky fields:

  • Application Name: You can put whatever you plan to call your personal setup (e.g., "My Bous Setup"). It does not need to be final.
  • Application URL: If you don't have a live project, you can use your GitHub profile URL, an empty repository, or simply https://localhost. TMDB accepts all of these for developer accounts.
  • Type of Use: Choose "Mobile Application", "Website", or whatever fits your use case best.
  • Application Summary: Be honest and upfront. We recommend something like:
    "A personal learning project — a movie discovery dashboard for educational and portfolio purposes only."
    The phrase “educational and portfolio purposes only” helps ensure instant approval!
Step 4 Copy your API Key
After submitting the form, you will be given your API keys. Look for the token labeled API Key (v3 auth) and copy it to your clipboard.
Step 5 Paste into Bous
Open the Bous extension in your browser, open the settings panel (gear icon), and paste your API Key into the TMDB API Key field. You're all set!
